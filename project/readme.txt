Databases Final Project - Spring 2017

Directions:
1. log into system user
2. run the load.sql query
	i. create.sql:
		creates or replaces the personal finance user,
		defines the directory,
		and connects to the personal finance user (BuyMe8)
	ii. schema.sql:
		calls drop.sql to drops all existing tables,
		creates, defines tables and adds constraints
	iii. data.sql:
		populates the database with starter data
	iv. triggers.sql:
		creates or replaces triggers
	v. procedures.sql:
		creates or replaces procedures
